# Footer CSS Documentation

## Core Styles

### Container (`.footer-container`)
- Flex column layout
- Centered content
- Light padding
- White text color

### Logo Styles (`.footer-logo`)
- 1.25rem font size
- Bold weight
- Small bottom margin

### Logo Container (`.footer-logo-container`)
- Circular background (--gray3)
- Fixed dimensions: 35x35px
- Centered content
- Padding: 0.5rem
- Margin adjustments

### Decorative Line (`.footer-line`)
- 90% width
- 5px height
- Dual gradient background
  - Uses `--gradient-line-1`
  - Uses `--gradient-line-2`
- Small vertical margins

## Dependencies
- CSS variables:
  - `--gray3`
  - `--gradient-line-1`
  - `--gradient-line-2`
